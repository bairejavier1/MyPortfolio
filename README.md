# My Portfolio

Welcome to my personal portfolio website! This project showcases my work, skills, and personal interests.

---

## 🔗 Live Demo

You can view the live version of this portfolio here:  
[https://bairejavier1.github.io/MyPortfolio/](https://bairejavier1.github.io/MyPortfolio/)

---

## 💻 Features

- **Responsive Design** – Works seamlessly on desktops, tablets, and mobile devices.  
- **Interactive Hero Section** – Animated typewriter effect for the logo text.  
- **Projects & Interests** – Cards displaying my projects and personal interests.  
- **Skills Grid** – Showcases my technical skills in a clean grid layout.  
- **Smooth Scrolling** – Easy navigation between sections using a fixed navbar.  
- **Contact Information** – Quick access to social media link.

---

## 🛠 Technologies Used

- **HTML5** – Semantic structure for all pages.  
- **CSS3** – Custom styling, media queries for responsive design, and smooth animations.  
- **JavaScript (ES6+)** – Typewriter effect, smooth scrolling, fade-in sections, and interactive elements.  
- **Git & GitHub Pages** – Version control and live deployment.
